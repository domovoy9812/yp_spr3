## Требования для запуска и настройка
### Для работы приложения требуется:
1. Java 21 версии
2. запущенный сервер Postgresql 17
+ должна быть создана основная база данных (по умолчанию **bliushtein_yp_sprint4_db**)
+ должен быть создан пользователь для подключения приложения к базе (по умолчанию **bliushtein_yp_sprint4**)
+ Для запуска интеграционных тестов также должна быть создана тестовая база данных (по умолчанию **bliushtein_yp_sprint4_test_db**)
### Настройка приложения
1. [application.yml](./web/src/main/resources/application.yml) Основной файл с настройками приложения

| параметр                                    | описание                                                                                                          | значение по умолчанию                                     |
|---------------------------------------------|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------|
| spring.datasource.url                       | строка подключения к БД                                                                                           | jdbc:postgresql://localhost:5432/bliushtein_yp_sprint3_db |
| spring.datasource.username                  | имя пользователя для подключения к БД                                                                             | bliushtein_yp_sprint3                                     |
| spring.datasource.password                  | пароль                                                                                                            | 12345                                                     |
2. [test-application.yml](./data/data-impl/src/test/resources/application-test.yml)

| параметр                                    | описание                                                                                                               | значение по умолчанию                                          |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| spring.datasource.url                       | строка подключения к тестовой БД                                                                                       | jdbc:postgresql://localhost:5432/bliushtein_yp_sprint3_test_db |
### Сборка и запуск
1. Команда для сборки проекта ```gradlew clean build```
2. Команда для запуска приложения ```gradlew bootRun```
3. После успешного запуска главная страница приложения будет доступна по ссылке http://localhost:8080/feed
